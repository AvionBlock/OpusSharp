Syntax

public int Decode(byte[] input, int length, short[] output, int frame_size, bool decode_fec)

Parameters

Name Type Description
input byte[] Input payload. Use null to indicate packet loss
length int Number of bytes in payload.
output short[] Output signal (interleaved if 2 channels). length is frame_size*channels*sizeof(short)
frame_size int Number of samples per channel of available space in pcm. If this is less than the maximum packet duration (120ms; 5760 for 48kHz), this function will not be capable of decoding some packets. In the case of PLC (data==NULL) or FEC (decode_fec=true), then frame_size needs to be exactly the duration of audio that is missing, otherwise the decoder will not be in the optimal state to decode the next incoming packet. For the PLC and FEC cases, frame_size must be a multiple of 2.5 ms.
decode_fec bool Request that any in-band forward error correction data be decoded. If no such data is available, the frame is decoded as if it were lost.

Return Value

Type Description
int Number of decoded samples or OpusErrorCodes.
